## 设计模式
### MVC
#### M(Model：操作数据库)
#### V(View：页面承载)
#### C(Controller：逻辑控制)

## 代码结构
### controllers
用于逻辑控制，介于view和modoel之间
### models
用来存储操作数据库
### utils
用于和三方的调用
### views
用于展示页面，包括html、js、css等前端相关内容
